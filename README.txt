Projeto de calculadora com interface grafica em python desenvolvida por Daniel Costa Carvalho Martins, como projeto da disciplina academica "Programação de computadores" do curso de Ciencias da computação do Centro Universitario de João Pessoa.

Tecnologias ultilizadas:
Python (Linguagem);
PySimpleGUI (biblioteca grafica);
cx_Freeze (biblioteca para converter .py em .exe);
Git/Github (versionamento de codigo e repositorio virtual);
Winrar (compactação de arquivos);
